## 🚀 Getting Started

### Three Simple Steps:

```bash
1. cd c:\Users\user\Desktop\chama
   composer install && npm install

2. cp .env.example .env
   php artisan key:generate
   [Edit .env with database credentials]

3. php artisan migrate
   php artisan db:seed
   npm run dev
   php artisan serve
```

**Visit:** http://localhost:8000
