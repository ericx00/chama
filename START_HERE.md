## ⚡ 3-Step Installation

### Step 1️⃣ Install Dependencies
```bash
cd c:\Users\user\Desktop\chama
composer install
npm install
```

### Step 2️⃣ Setup Environment
```bash
copy .env.example .env
php artisan key:generate
# Edit .env with your database
```

### Step 3️⃣ Initialize Database
```bash
php artisan migrate
php artisan db:seed
npm run dev
php artisan serve
```

✅ **Done!** Visit: http://localhost:8000

## 🚨 Troubleshooting Quick Fixes

| Problem | Solution |
|---------|----------|
| `php not found` | Add PHP to PATH |
| `Database error` | Check .env credentials |
| `Permission denied` | Check storage/ permissions |
| `npm not found` | Install Node.js |
| `Port 8000 busy` | Change port: `php artisan serve --port=8001` |
